# Тесты для лабораторной работы №01, задачи №07, варианта №01

## Входные данные
ЧПТ x - аргумент функции, eps - точность

## Выходные данные
ЧПТ s - приближённое значение, f - точное значение, 
delta - абсолютная погрешность, sigma - относительная погрешность (если существует)

## Позитивные тесты:
 - 01 - f(x) == 0, высокая точность;
 - 02 - f(x) == 0, низкая точность;
 - 03 - f(x) != 0;
 - 05 - eps < x, но eps большой (равен единице);

## Негативные тесты:
 - 01 - eps < 0;
 - 02-03 - ввод символа вместо числа (первого и второго соотв.);
 - 04 - недостаточно данные (одно число);
 - 05 - eps = 0;
