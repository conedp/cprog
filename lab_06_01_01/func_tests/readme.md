# Тесты для лабораторной работы №06, задачи №01, варианта №1

## Входные данные
Файл с информацией о фильмах в формате:
```
Название_1
Имя_режиссёра_1
Год_1
Название_2
...
```

## Выходные данные
В зависимости от аргументов командной строки:
1) Вывод информации о всех фильмах, отсортированных по выбранному полю
2) Вывод информации о найденном по выбранному полю и ключу фильме
