# Тесты для лабораторной работы №02, задачи №01, варианта №3

## Входные данные
Целое положительное - длина массива, целые - элементы массива

## Выходные данные
ЧПТ среднее геометрическое

## Позитивные тесты:
 - 01 - одно положительное: массив длины 1;
 - 02 - одно положительное в начале;
 - 03 - одно положительное в середине;
 - 04 - одно положительное в конце;
 - 05 - несколько положительных рядом;
 - 06 - несколько положительных не рядом;
 - 07 - несколько положительных рядом и не рядом;

## Негативные тесты:
 - 01 - размер массива = 0;
 - 02 - размер массива > 10;
 - 03 - размер массива < 0;
 - 04 - символ вместо длины массива;
 - 05 - символ в начале массива;
 - 06 - символ в середине массива;
 - 07 - символ в конце массива;
 - 08 - нет положительных элементов;
 - 09 - чисел в вводе меньше, чем задано;
