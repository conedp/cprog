# Тесты для лабораторной работы №03, задачи №03, варианта №3

## Входные данные
Целые положительные длина и ширина матрицы (<=10), целочисленная матрица, цифра

## Выходные данные
Изменённая целочисленная матрица

## Позитивные тесты:
Матрица 1 x K
 - 01 - матрица-строка, не изменяется;
Матрица 2 x K
 - 02 - строки упорядочены по возрастанию наименьшего, наименьшие на первом месте;
 - 03 - строки упорядочены по возрастанию наименьшего, наименьшие не на границах массива;
 - 04 - строки упорядочены по возрастанию наименьшего, наименьшие на последнем месте;
 - 05 - строки упорядочены по убыванию наименьшего, наименьшие на первом месте;
 - 06 - строки упорядочены по убыванию наименьшего, наименьшие не на границах массива;
 - 07 - строки упорядочены по убыванию наименьшего, наименьшие на последнем месте;
 - 11 - наименьее в обоих строках одинаковое (проверка на устойчивость сортировки);
Матрица N x M
 - 08 - матрица случайных чисел;
 - 09 - строки упорядочены по возрастанию наименьшего;
 - 10 - строки упорядочены по убыванию наименьшего;

## Негативные тесты:
 - 01 - символ вмместо длины матрицы;
 - 02 - символ вместо ширины матрицы;
Символ вместо элемента
 - 03 - в начале первой строки;
 - 04 - в середине первой строки;
 - 05 - в конце первой строки;
 - 06 - в начале k-й строки;
 - 07 - в середине k-й строки;
 - 08 - в конце k-й строки;
 - 09 - в начале последней строки;
 - 10 - в середине последней строки;
 - 11 - в конце последней строки;

 - 12 - длина матрицы равна 0;
 - 13 - длина матрицы < 0;
 - 14 - длина матрицы > 10;
 - 15 - ширина матрицы равна 0;
 - 16 - ширина матрицы < 0;
 - 17 - ширина матрицы > 10;
 - 18 - недостаточно чисел во вводе;
