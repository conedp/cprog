# Тесты для лабораторной работы №03, задачи №05, варианта №1

## Входные данные
Целые положительные длина и ширина матрицы (<=10), целочисленная матрица

## Выходные данные
Целочисленная матрица

## Позитивные тесты:
 - 01 - одно число, сумма цифр которого больше 10;
 - 02 - два положительных числа, сумма цифр которых больше 10;
 - 03 - два отрицательных числа, сумма цифр которых больше 10;
 - 08 - два подходящих числа, одно из которых - INT_MIN;
 - 04 - три числа, сумма цифр которых больше 10;
 - 05 - больше трёх чисел, сумма цифр которых больше десяти, расположенных рядом;
 - 06 - больше трёх чисел, сумма цифр которых больше десяти, расположенных не рядом;
 - 07 - больше трёх чисел, сумма цифр которых больше десяти, расположенных рядом и не рядом;

## Негативные тесты:
 - 01 - символ вмместо длины матрицы;
 - 02 - символ вместо ширины матрицы;
Символ вместо элемента
 - 03 - в начале первой строки;
 - 04 - в середине первой строки;
 - 05 - в конце первой строки;
 - 06 - в начале k-й строки;
 - 07 - в середине k-й строки;
 - 08 - в конце k-й строки;
 - 09 - в начале последней строки;
 - 10 - в середине последней строки;
 - 11 - в конце последней строки;

 - 12 - длина матрицы равна 0;
 - 13 - длина матрицы < 0;
 - 14 - длина матрицы > 10;
 - 15 - ширина матрицы равна 0;
 - 16 - ширина матрицы < 0;
 - 17 - ширина матрицы > 10;
 - 18 - недостаточно чисел во вводе;
 - 19 - нет чисел, сумма которых больше 10;
