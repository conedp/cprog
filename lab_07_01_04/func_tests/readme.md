# Тесты для лабораторной работы №07, сортировки №7, фильтрации №4

## Входные данные
Файл с массивом целых чисел

## Выходные данные
Файл с массивом целых чисел (отсортированных и, в случае указания флага, отфильтрованных)

## Позитивные тесты:
Без фильтрации
 - 01 - одно число;
 - 02 - два упорядоченных по возрастанию числа;
 - 03 - два упорядоченных по убыванию числа;
 - 04 - несколько упорядоченных по возрастанию чисел;
 - 05 - несколько неупорядоченных чисел;
 - 06 - несколько неупорядоченных чисел с дубликатами;
 - 07 - несколько упорядоченных в обратном порядке чисел с дубликатами;
С фильтрацией
 - 08 - два различных числа;
 - 09 - несколько различных чисел;
 - 10 - несколько различных чисел с дубликатами;

## Негативные тесты:
Без фильтрации
 - 01 - символ в вводе;
 - 02 - пустой файл;
С фильтрацией
 - 03 - одно число;
 - 04 - несколько одинаковых чисел;
Ошибки аргументов командной строки
 - 05 - меньше двух аргументов;
 - 06 - больше трёх аргументов;
 - 07 - нет входного файла;
 - 08 - третий аргумент != "f";
