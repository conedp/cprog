# Тесты для лабораторной работы №02, задачи №06, варианта №3

## Входные данные
Целые положительнаые длина матрицы и ширина матрицы (<=10) 

## Выходные данные
Изменённая целочисленная квадратная матрица

## Позитивные тесты:
 - 01 - матрица с длиной 1;
 - 02 - матрица с длиной 2;
 - 03 - матрица с чётной длиной > 2;
 - 04 - матрица с нечётной длиной > 1;
 - 05 - матрицы длины 10 (максимальной длины);

## Негативные тесты:
 - 01 - символ вместо длины матрицы;
 - 02 - символ вместо ширины матрицы;

 - 12 - длина матрицы равна 0;
 - 13 - длина матрицы < 0;
 - 14 - длина матрицы > 10;
 - 15 - ширина матрицы равна 0;
 - 16 - ширина матрицы < 0;
 - 17 - ширина матрицы > 10;
 - 18 - длина и ширина не совпадают;
