# Тесты для лабораторной работы №02, задачи №02, варианта №3

## Входные данные
Целое положительное - длина массива, целые - элементы массива

## Выходные данные
ЧПТ среднее геометрическое

## Позитивные тесты:
 - 01 - одно число Армстронга в начале;
 - 02 - одно число Армстронга в середине;
 - 03 - одно число Армстронга в конце;
 - 04 - массив из одного числа армстронга;
 - 05 - несколько чисел Армстронга рядом;
 - 06 - несколько чисел Армстронга не рядом;
 - 07 - несколько чисел Армстронга рядом и не рядом;

## Негативные тесты:
 - 01 - размер массива = 0;
 - 02 - размер массива > 10;
 - 03 - размер массива < 0;
 - 04 - символ вместо длины массива;
 - 05 - символ в начале массива;
 - 06 - символ в середине массива;
 - 07 - символ в конце массива;
 - 08 - нет чисел Армстронга элементов;
 - 09 - чисел в вводе меньше, чем задано;
