# Тесты для лабораторной работы №03, задачи №02, варианта №3

## Входные данные
Целые положительные длина и ширина матрицы (<=10), целочисленная матрица, цифра

## Выходные данные
Изменённая целочисленная матрица

## Позитивные тесты:
Матрица 1 x 1
 - 03 - число не содержит цифру;
Матрица 1 x K
 - 05 - нет чисел с данной цифрой;
 - 06 - одно число с данной цифрой в начале строки;
 - 07 - одно число с данной цифрой в середине строки;
 - 08 - одно число с данной цифрой в конце строки;
 - 09 - несколько чисел с данной цифрой;
Матрица N x M
 - 12 - нет столбцов с цифрой в числе;
 - 13 - один столбец с цифрой в одном числе;
 - 14 - один столбец с цифрой в нескольких числах;
 - 15 - один столбец с цифрой во всех числах;
 - 16 - несколько столбцов с цифрой в числах;

## Негативные тесты:
 - 01 - символ вмместо длины матрицы;
 - 02 - символ вместо ширины матрицы;
Символ вместо элемента
 - 03 - в начале первой строки;
 - 04 - в середине первой строки;
 - 05 - в конце первой строки;
 - 06 - в начале k-й строки;
 - 07 - в середине k-й строки;
 - 08 - в конце k-й строки;
 - 09 - в начале последней строки;
 - 10 - в середине последней строки;
 - 11 - в конце последней строки;

 - 12 - длина матрицы равна 0;
 - 13 - длина матрицы < 0;
 - 14 - длина матрицы > 10;
 - 15 - ширина матрицы равна 0;
 - 16 - ширина матрицы < 0;
 - 17 - ширина матрицы > 10;
 - 18 - недостаточно чисел во вводе;
 - 19 - цифра < 0;
 - 20 - цифра > 9;
 - 21 - символ вместо цифры;
Пустая матрица на выходе
Матрица 1 x 1
 - 22 - число === цифра;
 - 23 - число положительное содержит цифру;
 - 24 - число отрицательное и содержит цифру;
 - 26 - число - `INT_MIN`, цифра не равна `INT_MIN % 10`;
 - 28 - число - `INT_MIN`, цифра равна `INT_MIN % 10`;
Матрица 1 x K
 - 25 - все числа с данной цифрой;
Матрица N x M
 - 27 - все столбцы с цифрой в числах;
