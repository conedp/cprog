# Тесты для лабораторной работы №03, задачи №01, варианта №3

## Входные данные
Целые положительные длина и ширина матрицы (<=10), целочисленная матрица

## Выходные данные
Массив из 0 и 1

## Позитивные тесты:
 - 01 - матрица 1 x 1;
 - 02 - матрица k x 1;
 - 03 - матрица 1 x 2;
Матрица-строка
 - 04 - с возрастающей последовательностью;
 - 05 - с убывающей последовательностью;
 - 06 - с неубывающей последовательностью;
 - 07 - с невозрастающей последовательностью;
 - 08 - без монотонной последовательности;
 - 09 - из одинаковых чисел;

 - 10 - матрица, где все строки - монотонные последовательности;
 - 11 - матрица, где все строки - не монотонные последовательности;
 - 12 - матрица, где есть строки с монотонной последовательностью и без неё;

## Негативные тесты:
 - 01 - символ вмместо длины матрицы;
 - 02 - символ вместо ширины матрицы;
Символ вместо элемента
 - 03 - в начале первой строки;
 - 04 - в середине первой строки;
 - 05 - в конце первой строки;
 - 06 - в начале k-й строки;
 - 07 - в середине k-й строки;
 - 08 - в конце k-й строки;
 - 09 - в начале последней строки;
 - 10 - в середине последней строки;
 - 11 - в конце последней строки;

 - 12 - длина матрицы равна 0;
 - 13 - длина матрицы < 0;
 - 14 - длина матрицы > 10;
 - 15 - ширина матрицы равна 0;
 - 16 - ширина матрицы < 0;
 - 17 - ширина матрицы > 10;
 - 18 - недостаточно чисел во вводе;
