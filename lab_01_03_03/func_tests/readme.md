# Тесты для лабораторной работы №01, задачи №03, варианта №03

## Входные данные
ЧПТ v_0 - скорость, a - ускорение, t - время

## Выходные данные
ЧПТ s - перемещение

## Позитивные тесты:
 - 01 - все числа ненулевые;
 - 02 - ускорение равно 0;
 - 03 - начальная скорость равна 0;
 - 04 - ответ - нецелое число;
 - 05 - скорость и ускорение отрицательны;
 - 06 - скорость отрицательна, ускорение положительно;
 - 07 - скорость положительна, ускорение отрицательно;
 - 08 - время равно 0;
