# Тесты для лабораторной работы №01, задачи №05, варианта № 01

## Входные данные
Целые положительные a - делимое, d делитель

## Выходные данные
Целые положительные q - частное, r - остаток

## Позитивные тесты:
 - 01 - остаток равен нулю;
 - 02 - остаток не равен нулю;
 - 03 - a == d;
 - 04 - a < d;
 - 06 - делитель - единица;

## Негативные тесты:
 - 01 - не число на первой позиции;
 - 02 - не число на второй позиции;
 - 03 - a < 0;
 - 04 - d < 0;
 - 05 - a == 0;
 - 06 - d == 0;
 - 07 - один аргумент;
 - 08 - пустая строка;
 - 09 - числа, близкие к INT_MAX - проверка на переполнение (если int занимает 4 байта);
