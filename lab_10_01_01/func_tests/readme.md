# Тесты для лабораторной работы №10

## Описание задачи
Если ./app.exe file_name f to_find
то ищет структуру и выводит её или сообщение об отсутствии

Если ./app.exe file_name o
то сортирует и находит структуры, которые пролежали больше 365 дней (товары подлежат списанию)
и выводит их


## Входные данные
Файл с массивом структур, выбор операции, название товара (строка) для поиска в случае поиска

## Выходные данные
Найденная структура или структуры

## Позитивные тесты:
Поиск
 - 01 - запись не найдена;
 - 02 - запись в начале;
 - 03 - запись в конце;
 - 04 - запись в середине;
 - 05 - несколько записей;
Вывод товаров, пролежавших на складе больше 365 дней
 - 06 - 0 подходящих;
 - 07 - 1 подходящая запись;
 - 08 - несколько подходящих записей;
 - 09 - несколько одинаковых подходящих записей (одинаковых по количеству дней);
 - 10 - ровно 365 дней в записи;

## Негативные тесты:
Без фильтрации
 - 01 - несуществующий файл;
 - 02 - не "f" или "o" в аргументе-выборе операции;
 - 03 - нету строки для поиска;
 - 04 - меньше 3 аргументов;
 - 05 - больше 4 аргументов;
 - 06 - пустая строка в name;
 - 07 - не число в cost;
 - 08 - не число в n_days;
 - 09 - отрицательное число в cost;
 - 10 - отрицательное число в n_days;
