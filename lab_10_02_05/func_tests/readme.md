# Тесты для лабораторной работы №10, задача №2, варианту №5

## Входные данные
Строка-режим работа, строки для обработки (заканчивающиеся символом переноса строки)

## Выходные данные
Обработанные строки

## Позитивные тесты:
OUT
 - 01 - один символ;
 - 02 - строка длиной l: l % PART_LEN = PART_LEN - 1 (нуль-терминатор на последней позиции);
 - 03 - строка длиной l: l % PART_LEN = 0 (нуль-терминатор на первой позиции);
CAT
 - 04 - конкатенация двух символов;
 - 05 - сложение строк случайных длин;
 - 06 - сложение двух строк длиной l: l % PART_LEN = 0 (нуль-терминатор на первой позиции);
 - 07 - сложение двух строк длиной l1 и l2: (l1 + l2) % PART_LEN = 0 (нуль-терминатор на первой позиции);
 - 08 - сложение двух строк такой длины, что нуль-терминатор находится посередение узла;
SPS
 - 09 - 0 пробелов;
 - 10 - 1 двойной пробел;
 - 11 - несколько последовательностей пробелов разной длины;
 - 12 - пробелы в начале;
 - 13 - пробелы в конце;
POS
 - 14 - поиск смивола;
 - 15 - подстрока в начале строки;
 - 16 - подстрока в конце строки;
 - 17 - подстрока повторяется в строке;
 - 18 - подстрока не найдена;

## Негативные тесты:
 - 01 - неверное слово для выбора режима;
 - 02 - пустая строка для выбора режима;
 - 03 - пустая строка в списке;
 - 04 - пустая вторая строка при конкатенации;
 - 05 - пустая подстрока при поиске подстроки;
