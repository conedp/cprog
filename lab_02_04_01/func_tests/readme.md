# Тесты для лабораторной работы №02, задачи №01, варианта №3

## Входные данные
Целое положительное - длина массива, целые - элементы массива

## Выходные данные
ЧПТ среднее геометрическое

## Позитивные тесты:
 - 01 - вводится массив из одного элемента;
 - 02 - вводится массив из нескольких элементов (ввод заканчивается символом);
 - 03 - массив длиной ровно 10 (ввод заканчивается символом);
 - 04 - вводимый массив отсортирован;
 - 05 - вводимый массив отсортирован в обратную сторону;
 - 06 - в массиве присутствуют одинаковые элементы рядом;
 - 07 - в массиве присутствуют одинаковые элементы не рядом;
 - 08 - вводится отсортированный массив из двух элементов;
 - 09 - вводится обратно отсортированный массив из двух элементов;

## Негативные тесты:
 - 01 - не вводится ни одного числа;
 - 02 - вводится массив из 10 элементов (вводится больше 10, код возврата программы = 100);
