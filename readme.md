## Лабораторные работы по дисциплине "Программирование на C".   
Без условий заданий - требоваине кафедры

### Описание
Работы направлены на изучение основных аспектов языка C, классических типов и структур данных, процесса программирования, отладки и тестирования. 

### Структура
Каждая папка содержит локальную среду для отладки и тестирования в виде скриптов bash, написанные функциональные и модульные (не везде) тесты.  
Модульные тесты написаны с помощью CHECK.
